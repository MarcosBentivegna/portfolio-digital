# portfolio-digital

Portfolio personal de **Marcos Bentivegna** — Full Stack Developer y fundador de [Owtli](https://github.com/MarcosBentivegna).

Sitio estático con estética **neobrutalista** (azul y naranja), una sola página con secciones de experiencia, proyectos, skills y contacto.

## Vista previa

Abre `index.html` en el navegador o sirve la carpeta con un servidor local:

```bash
python3 -m http.server 8080
```

Luego visita `http://localhost:8080`

## Estructura

```
├── index.html          # Página principal
├── support.js          # Runtime dc
├── uploads/            # Imágenes de proyectos y CV
└── screenshots/        # Capturas de referencia
```

## Stack

- HTML + CSS (inline)
- dc-runtime (`x-dc`)
- Google Fonts: Syne, Space Grotesk
